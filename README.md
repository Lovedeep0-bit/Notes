# Notes 📝

A sleek, secure, and minimal note-taking app built with modern Android standards.

## Features ✨

- **Markdown Support**: Take rich notes with support for **bold** and <u>underlined</u> text.
- **Biometric Security**: Protect your privacy with fingerprint or device credential locking. Includes a smart 5-minute grace period.
- **Notebooks & Organization**: Group your notes into notebooks or keep them pinned for quick access.
- **Checklists**: Integrated task management with interactive checkboxes in both preview and edit modes.

## Tech Stack 🛠️

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/compose)
- **Local Database**: [Room](https://developer.android.com/training/data-storage/room)
- **Authentication**: [AndroidX Biometric](https://developer.android.com/training/sign-in/biometric-auth)
- **Navigation**: Compose Navigation
